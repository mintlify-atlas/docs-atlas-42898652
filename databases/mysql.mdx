<Warning>
  MySQL support is **not yet available**. The driver is in development and has not been released.
</Warning>

MySQL is on the Dory roadmap. Once released, it will follow the same connection model as [PostgreSQL](/databases/postgresql) and [ClickHouse](/databases/clickhouse) — you will be able to add a MySQL connection, browse your schema, and use the SQL editor and AI tools against your MySQL databases.

## What to expect when MySQL support arrives

Based on Dory's existing database support model, MySQL will include:

- **SQL editor** — multi-tab workspace with schema-aware autocomplete
- **SQL Copilot** — AI-assisted query generation, fixing, and explanation grounded in your MySQL schema
- **Database chatbot** — conversational exploration of your tables and data
- **Schema explorer** — browse databases, tables, and columns in the sidebar
- **Connection management** — host, port, database, username, password, and SSL configuration

Deep operational features (monitoring, privilege management) are ClickHouse-specific and are not planned for MySQL.
