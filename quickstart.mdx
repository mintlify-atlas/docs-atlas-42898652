## Choose your deployment

Dory runs as a Docker container or as a native desktop application. Pick the option that fits your workflow.

<CardGroup cols={2}>
  <Card title="Docker" icon="docker" href="/deployment/docker">
    Deploy on your own infrastructure. Best for teams and self-hosted setups.
  </Card>
  <Card title="Desktop app" icon="monitor" href="/deployment/desktop">
    Download and run locally on macOS or Windows. No server required.
  </Card>
</CardGroup>

## Get up and running

<Steps>
  <Step title="Run Dory">
    The fastest way to start is with Docker. Make sure Docker is installed, then run:

    ```bash
    docker run -d --name dory \
      -p 3000:3000 \
      -e DS_SECRET_KEY="$(openssl rand -base64 32 | tr -d '\n')" \
      -e BETTER_AUTH_SECRET="$(openssl rand -hex 32)" \
      -e BETTER_AUTH_URL="http://localhost:3000" \
      -e DORY_AI_PROVIDER=openai \
      -e DORY_AI_MODEL=gpt-4o-mini \
      -e DORY_AI_API_KEY=your_api_key_here \
      -e DORY_AI_URL=https://api.openai.com/v1 \
      dorylab/dory:latest
    ```

    For full configuration options and environment variable details, see the [Docker deployment guide](/deployment/docker).

    Prefer a native app? [Download the desktop app](/deployment/desktop) for macOS or Windows instead.
  </Step>

  <Step title="Open Dory in your browser">
    Once the container is running, open your browser and navigate to:

    ```
    http://localhost:3000
    ```

    <Note>
      The desktop app opens automatically after launch — no browser step needed.
    </Note>
  </Step>

  <Step title="Add a database connection">
    After Dory loads, add your first database connection. See [Adding a connection](/connections/adding-connection) for step-by-step instructions.

    Dory currently supports:

    | Database   | Status               |
    |------------|----------------------|
    | ClickHouse | Deeply integrated    |
    | PostgreSQL | Supported (Beta)     |
    | MySQL      | In development       |
    | DuckDB     | Planned              |
    | SQLite     | Planned              |
  </Step>

  <Step title="Run your first query">
    Open the SQL editor from the left sidebar. Start typing — Dory provides schema-aware autocomplete based on your connected database.

    To use the AI Copilot, type a natural language question in the Copilot panel and Dory will generate SQL grounded in your real schema.

    <Tip>
      You can also open the database chatbot to ask questions about your tables and data in plain language.
    </Tip>
  </Step>
</Steps>
