## What is Dory?

Dory is a full-featured data studio that runs as a **Docker container** or a **native desktop application** (macOS and Windows). It connects to your existing databases and layers AI-powered tools on top of them — without requiring you to move data anywhere.

## Deployment options

Dory supports two runtime modes:

- **Docker** — Self-host with a single `docker run` command. Ideal for teams and server deployments.
- **Desktop** — Native macOS and Windows app. Ideal for individual developers and analysts.

Both modes use the same core application — the runtime is determined by the `DORY_RUNTIME` environment variable.

## Who is Dory for?

- **Data engineers** building and debugging data pipelines
- **Data analysts** exploring and querying databases
- **Database platform teams** managing ClickHouse clusters
- **ClickHouse operations teams** monitoring query performance and managing access control
